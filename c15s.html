Пользовательская установка C15S Sub

Цена по запросу (₽)

Стремясь к тому, чтобы наши ЛУЧШИЕ «универсальные колонки» подходили для домашнего кинотеатра, мы разработали настоящую басовую машину без компромиссов.

Представляем сабвуфер С15S для индивидуальной установки. Предназначен для индивидуальной установки:

• Чем больше, тем лучше, когда дело касается басов. C158, занимающий промежуточное положение между Х10 и Х12 по общей площади низкочастотного динамика, — это бескомпромиссная басовая машина.
• Модель С15S, предназначенная для установки за экраном, имеет глубину всего 298 мм, как и S150.
• Полностью изготовленный из МДФ корпус толщиной 1 дюйм с центральной распоркой и дополнительными распорками спереди и сзади для ультрапрочной конструкции корпуса.
• В комплект входят кронштейны Bass Brace для крепления к полу/сцене для обеспечения безопасности и снижения вибрации.
• В комплект входит соединительный элемент для объединения нескольких сабвуферов для увеличения мощности.
• Отдельные сабвуферы и усилители повышают тепловые характеристики.
• Внешний усилитель — VA500 Blue, настроенный специально для C15S с 3 пользовательскими эквалайзерами.

С15S создан на основе технологии сабвуферов Х-Series и V-Series и отличается большими размерами. Обладая мощностью, сопоставимой с мощностью двух 10-дюймовых сабвуферов Х-10 (без преимуществ технологии Push-Pull) в одном корпусе, С15S предназначен для индивидуальных и масштабных инсталляций.

С использованием специально настроенного усилителя VА-500 вся система была разработана с нуля для воспроизведения самых низких нот на таком уровне громкости, чтобы их можно было услышать и почувствовать.

Не нужно много времени, чтобы понять, что это очень круто и, прежде всего, серьёзно.
Журнал Hemmablo (Домашее кино), Швеция, май 2016
